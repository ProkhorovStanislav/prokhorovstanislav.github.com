
Необходимо сделать форму, состоящую из пяти полей:
ФИО (текст)
Телефон (поле с маской по телефону +7(___)___-__-__)
E-mail (текст)
Город (текст)
Адрес доставки (текст)

Форма должна быть адаптивной и адекватно отображаться на разных устройствах. Особенности логики формы:
Все пять полей формы должны валидироваться “на лету”.
Поле ФИО обычный текст
Поле телефон должно быть с маской для соответствия телефону. Начинать ввод можно только с первого незаполненного символа, то есть нельзя начать ввод с середины номера, только с первого незаполненного символа.
Поле e-mail должно валидироваться таким образом, чтобы проходили только похожие на реальность почтовые ящики :)
Поле город должно работать с автокомплитом. Список городов можно получать через AJAX по ссылке http://planeta-sport.ru/bitrix/components/bitrix/sale.ajax.locations/search.php?search=<запрос>&params=siteId:s1 Пример: http://planeta-sport.ru/bitrix/components/bitrix/sale.ajax.locations/search.php?search=Санк&params=siteId:s1
Адрес доставки - обычное поле не менее 10-и символов.
Если какое-либо поле не заполнено, либо заполнено неверно, то должно появляться предупреждение под самим полем.

Дополнительным плюсом будет:
Отображение пола после ввода ФИО.
Появление карты с отметкой после выбора города
